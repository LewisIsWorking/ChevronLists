# Step 2 — Add Markers

Add any of these directly inside item content:

| What you type | What it does |
|---|---|
| `[ ]` or `[x]` | Checkbox — toggle with `CL: Toggle Item Done` |
| `#tag` | Tag — autocomplete shows existing tags |
| `!!!` | Urgent (red background) |
| `@2026-04-15` | Due date — overdue items show `⚠` in the status bar |
| `~2h` | Time estimate |
| `+3` | Vote count |

**Example:**
```markdown
>> - !!! Fix the login bug #backend @2026-04-01 ~1h
```
