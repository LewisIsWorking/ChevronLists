# Step 5 — Kanban, Stats & Export

**Views:**
- `CL: Show Kanban` — items grouped as Todo / In Progress (starred) / Done
- `CL: Today View` — all overdue and due-today items across the workspace
- `CL: Show Tag Stats` — completion % per tag in a webview table
- `CL: Show Section Growth` — bar chart of item count per section

**Statistics:**
- `CL: Quick Stats` — one-line summary of the current section
- `CL: Show Mentions Report` — `@Name` assignment table with done/total
- `CL: Show Section Time Estimate` — sum of all `~estimate` markers

**Export:**
- `CL: Copy Section As…` — Markdown, Plain Text, JSON, CSV, or HTML
- `CL: Export to Obsidian` — YAML frontmatter with `##` sections and emoji markers

**AI Assist** (requires `chevron-lists.anthropicApiKey`):
- `CL: Suggest Items (AI)` · `CL: Expand Item (AI)` · `CL: Rewrite Item (AI)`
