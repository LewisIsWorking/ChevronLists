# Step 4 — Live Decorations

Decorations update automatically as you type:

- **Section Summary** — `5 items · ▁▃▅░ 2 done · 3 tags` ghost text after every header
- **Progress Bar** — `▓▓▓░░ 3/5` for sections with checkboxes
- **Word Goal Bar** — add `==500` to a header for a live word count goal
- **Priority backgrounds** — `!!!` items glow red, `!!` amber, `!` yellow
- **⏰ due soon** — inline ghost text on items within 3 days
- **Sticky Header** — section name stays visible at top of viewport when scrolled

Toggle any decoration with `CL: Toggle Section Summary`, `CL: Toggle Checklist Progress Bar`, etc.
