# Step 3 — Navigate and Filter

With multiple sections in a file, use:

- **`Ctrl+Alt+Down`** / **`Ctrl+Alt+Up`** — jump between section headers
- **`CL: Filter Sections`** — fuzzy-search jump to any section in the file
- **`CL: Filter by Tag`** — show only items with a chosen `#tag`
- **`CL: Today View`** — all items due today or overdue, across every file

Type `CL:` in the Command Palette (`Ctrl+Shift+P`) to see every available command.
